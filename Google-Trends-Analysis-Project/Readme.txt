# Google Trends Market Intelligence & Forecasting

## Project Overview
This project analyzes Google search trends using Python and the Pytrends API. It provides insights into technology trends such as Artificial Intelligence, Data Science, Machine Learning, Cloud Computing, and Cybersecurity.

## Objectives
- Analyze search trends over time
- Compare multiple technologies
- Identify regional interest
- Discover related search queries
- Forecast future trends

## Technologies Used
- Python
- Pandas
- Pytrends
- Matplotlib
- Seaborn
- Prophet

## Project Structure
notebook - Jupyter notebook with analysis  
data - extracted Google Trends data  
outputs - generated charts  
presentation - project PPT  

## Results
The project identifies trending technologies and predicts future search demand using time series forecasting.
